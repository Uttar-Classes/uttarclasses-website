# Uttar Classes Website - Cleaned UI Code

Static website for https://www.uttarclasses.in.

## What was cleaned

- Rebuilt shared CSS for consistent desktop and mobile layout.
- Fixed mobile navigation behavior.
- Removed placeholder/fake sections from About and Results pages.
- Removed broken competitive-result image references because the repo currently has only Class 10 and Class 12 result images.
- Replaced inline styles with reusable CSS.
- Fixed WhatsApp enquiry form encoding and validation.
- Updated sitemap to include Subhanpura and Bhayli pages.
